# Omni Sample
The following iPhone app is part of my application to join the Omni team

### Requirements
   1. XCode 8.2
   2. iOS 10
   3. Cocoapods
   
### Installation
   1. cd into the directory with `Podfile` in terminal
   2. run `pod install`
   3. open `Omni Sample.xcworkspace` in Xcode
   4. Run!
